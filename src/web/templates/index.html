초고정밀 멀티랭귀지 OCR 시스템

일본 비즈니스 문서를 위한 초정밀 OCR 시스템으로, 다양한 OCR 엔진을 앙상블하여 99.5% 이상의 정확도를 제공합니다.

주요 기능

다국어 지원

일본어, 한국어, 영어, 중국어(간체/번체) 등 다양한 언어를 지원합니다.

자동 방향 보정

기울어지거나 회전된 페이지를 자동으로 감지하고 보정합니다.

고정확도 OCR

여러 OCR 엔진 앙상블로 최고 수준의 정확도를 제공합니다.

특수 항목 인식

도장, 손글씨, 수정된 텍스트 등 특수 항목을 인식합니다.

LLM 기반 추출

설정 가능한 필드를 LLM을 활용하여 자동으로 추출합니다.

CSV 내보내기

추출된 데이터를 CSV 형식으로 손쉽게 내보낼 수 있습니다.

시스템 아키텍처

시작하기

  1. 문서 업로드 - "문서 업로드" 버튼을 클릭하여 PDF 또는 이미지 파일을 업로드합니다.
  2. 처리 대기 - OCR 처리가 완료될 때까지 대기합니다. 처리 상태를 실시간으로 확인할 수 있습니다.
  3. 결과 확인 - 처리 완료 후 텍스트 추출 결과를 확인합니다.
  4. 필드 추출 - "데이터 추출" 탭에서 추출된 key-value 데이터를 확인하고 편집합니다.
  5. CSV 내보내기 - "내보내기" 버튼으로 추출 데이터를 CSV로 저장합니다.
